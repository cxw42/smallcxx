◆ setLogLevel()

void setLogLevel ( LogLevel  newLevel,
const std::string &  domain = SMALLCXX_DEFAULT_LOG_DOMAIN 
)

Set log level for domain to newLevel.

Parameters
[in]newLevel- New level. Must be LOG_SILENT, or between LOG_MIN and LOG_MAX (inclusive). In particular, you may not set the level to LOG_PRINT or LOG_PRINTERR.
[in]domain- Which log domain to apply newLevel to.
Warning
Some log messages may be dropped when the level changes.
Exceptions
std::runtime_errorif you try to set LOG_PRINT or LOG_PRINTERR.

◆ setVerbosityFromEnvironment()

void setVerbosityFromEnvironment ( const char *  detailEnvVarName = nullptr)

Set the verbosity for all log domains based on $V.

If $V exists and is a decimal >=1, the system level is set to INFO + $V (e.g., 1 = LOG_DEBUG, 2 = LOG_LOG).

Note
Does not set verbosities for all log domains.
Todo:
implement a GST_DEBUG-style log-level selector.
Warning
Must be called before any messages are logged if it is to affect all log domains.
Parameters
[in]detailEnvVarName- if given, non-NULL, and nonempty, the name of a GStreamer-style log-level variable to be used instead of $V if the given variable exists.
