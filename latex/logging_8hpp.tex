\doxysubsection{Detailed Description}
Header for logging library. 

\begin{DoxyAuthor}{Author}
Christopher White \href{mailto:cxwembedded@gmail.com}{\texttt{ cxwembedded@gmail.\+com}} 
\end{DoxyAuthor}
\begin{DoxyCopyright}{Copyright}
Copyright (c) 2021 Christopher White
\end{DoxyCopyright}
Terminology\+: messages are logged if the \char`\"{}message level\char`\"{} is less than or equal to the \char`\"{}system level\char`\"{}.

There can be any number of log-\/message domains, each identified by an arbitrary string. Strings starting with {\ttfamily \char`\"{} \char`\"{}} (a space) are reserved for use by smallcxx/logging. To define your own domain for messages in a source file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#define SMALLCXX\_LOG\_DOMAIN "myDomain"}}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <smallcxx/logging.hpp>}}
\end{DoxyCode}
