\doxysubsection{Detailed Description}
A matcher holding any number of glob patterns, include or exclude. 

The patterns are grouped into sets, each glob in a set having the same Polarity. For example, 
\begin{DoxyCode}{0}
\DoxyCodeLine{*.bak}
\DoxyCodeLine{*.swp}
\DoxyCodeLine{!*.foo}
\DoxyCodeLine{*.bar}
\end{DoxyCode}


will have, in order, a \mbox{\hyperlink{namespacesmallcxx_1_1glob_a1caf8d7a1114eb187ef69330520bb5a3af4d72a64acd8929c0cc9ed96a7a336cc}{Polarity\+::\+Include}} set matching {\ttfamily $\ast$.\{bak,swp\}}; a \mbox{\hyperlink{namespacesmallcxx_1_1glob_a1caf8d7a1114eb187ef69330520bb5a3a843f2812f595e7ec7c5036e89fde02d6}{Polarity\+::\+Exclude}} set matching {\ttfamily $\ast$.foo}; and a \mbox{\hyperlink{namespacesmallcxx_1_1glob_a1caf8d7a1114eb187ef69330520bb5a3af4d72a64acd8929c0cc9ed96a7a336cc}{Polarity\+::\+Include}} set matching {\ttfamily $\ast$.bar}.

All globs follow the \href{https://editorconfig.org}{\texttt{ Editor\+Config}} format. \begin{DoxyNote}{Note}
Path entries are separated by {\ttfamily /} (forward slash) on all platforms! 
\end{DoxyNote}
