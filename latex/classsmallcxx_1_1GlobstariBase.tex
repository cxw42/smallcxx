\doxysubsection{Detailed Description}
S\+A\+X-\/style directory searcher\+: base class. 

Subclass this and implement the virtual functions to do your own traversing. All globs follow the \href{https://editorconfig.org}{\texttt{ Editor\+Config}} format. \begin{DoxyNote}{Note}
Path entries are separated by {\ttfamily /} (forward slash) on all platforms!
\end{DoxyNote}
\char`\"{}\+Globstari\char`\"{} = supports {\itshape glob}, glob\+\_\+star\+\_\+, and \+\_\+i\+\_\+gnores.
