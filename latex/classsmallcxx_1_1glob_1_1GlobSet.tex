\doxysubsection{Detailed Description}
A set of globs. 


\begin{DoxyItemize}
\item Each glob must match against the entire string. E.\+g., {\ttfamily $\ast$.txt} will match {\ttfamily foo.\+txt} but not {\ttfamily foo/bar.\+txt}.
\item Glob checks do not treat dot files specially. Therefore, {\ttfamily $\ast$foo} will match {\ttfamily foo} and {\ttfamily .foo}.
\item No globset matches an empty string.
\end{DoxyItemize}

All globs follow the \href{https://editorconfig.org}{\texttt{ Editor\+Config}} format. \begin{DoxyNote}{Note}
Path entries are separated by {\ttfamily /} (forward slash) on all platforms! 
\end{DoxyNote}
