Detailed Description

SAX-style directory searcher: base class.

Subclass this and implement the virtual functions to do your own traversing. All globs follow the EditorConfig format.

Note
Path entries are separated by / (forward slash) on all platforms!

"Globstari" = supports glob, glob_star_, and _i_gnores.

Returns a list of ignore files to load, if they exist, for dirName.

Parameters
[in]dirName- the canonical path to the directory
Returns
A list of zero or more ignore paths, absolute or relative.-
  • If absolute, that file is checked.
  • If relative, that file is checked in dirName. Therefore, you can return a fixed vector if you only have relative-path ignore files.

The default ignore is .eignore in the current directory. The e in .eignore is because it uses EditorConfig-style globs.

Find files in basePath matching needle.

Parameters
[in]basePath- where to start. This does not have to be a directory on disk.
[in]needle- EditorConfig-style globs indicating the files to find. These are with respect to basePath.
[in]maxDepth- maximum recursion depth. -1 for unlimited.
Note

  • Glob checks do not treat dot files specially. Therefore, *foo will match foo and .foo.
  • All glob checks are done against canonicalized paths. Therefore, **‍/* will match everything.
  • traverse() will always call processEntry on the starting dir.
