Detailed Description

A set of globs.

  • Each glob must match against the entire string. E.g., *.txt will match foo.txt but not foo/bar.txt.
  • Glob checks do not treat dot files specially. Therefore, *foo will match foo and .foo.
  • No globset matches an empty string.

All globs follow the EditorConfig format.

Note
Path entries are separated by / (forward slash) on all platforms!
